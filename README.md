# 🌐 Personal Portfolio Website  

## 📌 Project Overview  
This is a **personal portfolio website** created using **HTML, CSS, and JavaScript**.  
It showcases my skills, projects, and achievements with a clean, responsive, and user-friendly design.  
This project was built to strengthen my frontend development skills and create a professional online presence.  

---

## 🚀 Features  
- Responsive design for mobile, tablet, and desktop  
- Interactive navigation menu and smooth scrolling  
- Sections for **About Me, Skills, Projects, Contact**  
- Custom CSS animations and transitions  
- Contact form (using basic JavaScript validation)  

---

## 📚 Learnings  
Through this project, I learned:  
- Structuring and organizing a website with **semantic HTML**  
- Styling with **CSS3** including Flexbox/Grid for responsive layouts  
- Adding interactivity with **JavaScript** (menus, animations, form validation)  
- Optimizing website performance and accessibility  
- Designing a consistent **UI/UX** from scratch  

---

## 💡 What I Liked the Most  
I especially enjoyed designing the **user interface and animations** to make the site more engaging and interactive. This project helped me sharpen my **frontend development and creative design** skills.  

---

## 🛠️ Tech Stack  
- **Frontend:** HTML5, CSS3, JavaScript  
- **No frameworks used**
